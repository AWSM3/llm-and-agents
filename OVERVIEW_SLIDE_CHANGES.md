# Добавлен новый слайд "Про ИИ" (OverviewSlide)

## Описание

Создан новый первый слайд презентации, который предшествует слайду "Что меняется?" (IntroSlide). Слайд представляет обзорную информацию о развитии и текущем состоянии ИИ-моделей.

## Изменённые файлы

### 1. `components/slides/OverviewSlide.tsx` (новый файл)
Компонент нового слайда с разделами:
- Заголовок и подзаголовок
- Эволюция ИИ (timeline с основными игроками)
- Текущее состояние (количество моделей и новые возможности)
- Вызовы и возможности
- Полезные ресурсы со ссылками

### 2. `lib/content-data.ts`
Добавлены:
- Интерфейсы: `OverviewMilestone`, `OverviewChallengePoint`
- Секция `overview` в `ContentData.sections`
- Данные для слайда в объекте `contentData`
- Обновлена навигация с добавлением пункта "Про ИИ"

### 3. `app/page.tsx`
- Импортирован `OverviewSlide`
- Обновлён массив `SLIDE_IDS` с добавлением 'overview'
- Добавлен рендер слайда на позиции 0
- Сдвинуты индексы последующих слайдов (intro, models, engineering, tools, inference)

## Содержание слайда

Слайд лаконично представляет:

1. **Эволюция ИИ**: От OpenAI до китайских моделей (DeepSeek, Qwen, GLM)
2. **Актуальные модели**: 10+ топовых моделей (ChatGPT 5, Claude Sonnet 4.5, Gemini 3 Pro и др.)
3. **Новые возможности**: веб-поиск, function calling, reasoning, большие контексты
4. **Вызовы**: переход от ручной обработки информации к использованию мощных ИИ-инструментов
5. **Ресурсы**: ссылки на Wikipedia, OpenRouter, Life Architect

## Порядок слайдов

1. **Hero** (главная)
2. **Про ИИ** (новый - OverviewSlide) ← 
3. **Что меняется?** (IntroSlide)
4. **Модели** (ModelsSlide)
5. **Инженерия** (EngineeringSlide)
6. **Инструменты** (ToolsSlide)
7. **Провайдеры** (InferenceSlide)

## Стиль и дизайн

Слайд следует существующему дизайну презентации:
- Использует компоненты из shadcn/ui (Card, Badge)
- Применяет анимации с задержками (`slide-enter`)
- Адаптивный layout (grid responsive)
- Иконки emoji для визуальной идентификации
- Gradient для акцентного блока с вызовами
